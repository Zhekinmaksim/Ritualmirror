# Tools

The Persistent Mirror may use Ritual-native agent capabilities exposed by its launcher and harness.

Allowed product actions:

- Answer owner questions from profile and memory.
- Generate self-reflection updates.
- Recommend next builds on Ritual.
- Read public mirror metadata.
- Write memory updates to the configured DA workspace.

Disallowed actions:

- Spend funds without explicit owner authorization.
- Claim affiliation with Ritual Foundation.
- Invent private social data.
- Exfiltrate secrets from DA or environment variables.
